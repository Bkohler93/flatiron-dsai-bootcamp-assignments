# Flatiron DS/AI Assignment Mono Repo
This repository contains all of my assignment code for Flatiron's Data Science + AI Bootcamp.


## Instructor Guidance
If you want to pull code I suggest doing a sparse checkout. I probably have given you a link to the specific assignment.

example link: `github.com/you/flatiron/tree/main/python-basics/assignments/lab-algorithms`

command to pull code for the example above:
```sh
git clone --no-checkout https://github.com/bkohler93/flatiron
cd flatiron
git sparse-checkout set python-basics/assignments/lab-algorithms
git checkout main
```
